Secret Scanner

Test Github repositories for potential secret/key leaks

A project for Microblink internship. Main features developed by Microblink, additional features added by Filip Matić

This small app has been developed by Filip Matić as an application for a Microblink Software development internship. Most of the backend has been developed by the Microblink, but this simple UI and additional backend features are added as a part of the technical testing.
The Secret Scanner is able to go through a Github repository and see if you have any "secrets" mistakenly left visible, such as API keys, passwords and similar. After being able to access public repositories, now it can access private repos also by ...
Also, by connecting to an outside LLM service, app now suggests how to fix this secret leaking for your safety and privacy reasons.
